SATIVA O INDICA?

Sativa

La planta de cannabis Sativa es una variedad de cannabis que se cree que tiene un alto contenido de THC y un bajo contenido de CBD. Esta variedad de cannabis se cree que es más psicoactiva que el cannabis Indica, pero se cree que tiene un mayor efecto energizante. Se cree que la planta de cannabis Sativa se usa para tratar la fatiga, el estrés, la depresión y el dolor crónico.

Indica

La planta de cannabis Indica es una variedad de cannabis que se cree que tiene un alto contenido de CBD y un bajo contenido de THC. Esta variedad de cannabis se cree que es menos psicoactiva que el cannabis Sativa, pero se cree que tiene un mayor efecto relajante. Se cree que la planta de cannabis Indica se usa para tratar la ansiedad, el insomnio, el dolor crónico y la inflamación.

LAS PLANTAS SATIVAS MAS FUERTES

El CBD (cannabidiol) es un cannabinoide natural que se encuentra en la planta de cannabis. Se cree que el CBD puede tener una variedad de beneficios para la salud, como el alivio del dolor crónico, la reducción de la ansiedad y la reducción de la inflamación. El CBD también se ha utilizado para tratar la epilepsia, el glaucoma y otros problemas de salud.
